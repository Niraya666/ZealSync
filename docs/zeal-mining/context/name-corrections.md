## 特殊格式处理

| 原始格式 | canonical_name | 处理方式 |
|----------|----------------|----------|
| `@&` | `&` | 符号昵称，原样保留 |
| `@Liqp` | `Liqp` | 仅昵称，无行业信息 |
| `@Eva（社群发起人）` | `Eva` | 括号内为备注，不作为昵称一部分 |

## 跨期一致性

- Eva 在所有期中的 canonical_name 统一为 "Eva"
