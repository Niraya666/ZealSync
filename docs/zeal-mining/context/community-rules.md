# Community Rules — ZealSync Specific

> 本文件包含 ZealSync 社群的特定规则，供 zeal-mining skill 在运行时查阅。
> Skill 本身不包含这些具体信息，以保证可迁移性。

## 角色定义

### 主持人 (Host)
- **Eva** 在所有期中担任主持人
- 主持人负责开场、引导流程、总结
- 识别关键词："主持人"、"Eva 开场"、"Eva 总结"

### 领航员 (Navigator)
- 每期 1-2 人，负责引导当期话题
- 识别关键词："领航员" + 人名，如 "领航员三金率先介绍"
- 领航员通常在开场时做主题分享

### 参与者 (Participant)
- 默认角色，除主持人和领航员外的人员

## 格式规范

### 小作文标题格式
```
@昵称-行业-岗位：
@昵称：
@昵称（备注）：
```

### 会议纪要结构
- "参会人员自我介绍" —— 含实名和背景
- "智能章节" —— 含说话人编号和发言内容
- "关键决策" —— 含会议决策
- "金句时刻" —— 含引用和说话人
