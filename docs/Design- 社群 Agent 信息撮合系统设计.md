**核心场景**：

- **组队**：项目合作、活动组织、兴趣小组
- **Coffee-chat**：经验交流、职业咨询、社交破冰
- **信息匹配**：基于技能、兴趣、目标的人群推荐

**用户旅程**：

```
## 信息获取和上传阶段
1. Onboarding：对话式信息收集（你是谁、想找什么、能提供什么）
2. 信息上传：文档/blog/社交媒体账号（可选）
3. Agent 理解：解析信息，构建个人画像
4. HITL：检查确定，提交
   ## 
5. 匹配推荐：Agent 自动发现合适人群
6. 双向确认：双方 Agent 交换信息，人类确认
7. 促成连接：提供联系方式/安排会议
```
