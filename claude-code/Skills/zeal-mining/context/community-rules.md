# Community Rules — Template

> 本文件为模板，使用时请复制到工作路径并填入社群特定信息。
> Skill 运行时会读取工作路径下的实际文件，而非本模板。

## 角色定义

### 主持人 (Host)
- **{主持人昵称}** 在所有期中担任主持人
- 主持人负责开场、引导流程、总结
- 识别关键词："主持人"、"{主持人姓名} 开场" 等

### 领航员 (Navigator)
- 每期 1-2 人，负责引导当期话题
- 识别关键词："领航员" + 人名
- 领航员通常在开场时做主题分享

### 参与者 (Participant)
- 默认角色，除主持人和领航员外的人员

## 格式规范

### 小作文标题格式
```
@昵称-行业-岗位：
@昵称：
@昵称（备注）：
```

### 会议纪要结构
- "参会人员自我介绍" —— 含实名和背景
- "智能章节" —— 含说话人编号和发言内容
- "关键决策" —— 含会议决策
- "金句时刻" —— 含引用和说话人
